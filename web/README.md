## Install
```bash
// install dependencies
npm install
```
## Run

切换到 node 14版本
sudo n 14

npm install style-loader css-loader style-loader --save-dev

npm install less less-loader --save-dev

npm install sass sass-loader --save-dev

npm install node-sass --save


### Development
```bash
npm run dev
```
### Production(Build)
```bash
npm run build
```
